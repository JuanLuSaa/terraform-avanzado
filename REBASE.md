# Proceso de Rebase Interactivo

1. **Objetivo**: Limpiar los commits de prueba generados durante la configuración de LocalStack.
2. **Comando**: Usé `git rebase -i HEAD~N` para agrupar 3 commits.
3. **Acciones**: Apliqué `squash` para fusionar commits y `reword` para mejorar los mensajes.
4. **Resultado**: Un historial de Git limpio, legible y profesional.
